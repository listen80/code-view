# codejs

## 安装
1. npm install
2. npm start

## 用法
### 引入codejs
<script type="code.js"></script>
### xmp,script标签上加code属性

```
<xmp code>
<html>
  <body>
    <div>
      <p>test</p>
    </div>
  </body>
<html>
</xmp>
```

```
<script code type="js">
  var a = 10;
  var b = function() {}
</script>
```
